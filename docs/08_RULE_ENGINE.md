# Rule Engine

## Purpose

Provide smart farming suggestions without AI.

## Examples

### Seasonal Advice

IF month = June–September
→ Show "SW Monsoon advice"

### Disease Handling

IF problem = Azhukal
→ Suggest:

* Improve drainage
* Apply fungicide
* Remove infected plants

### Growth Stage Tips

IF stage = Flowering
→ Show:

* Nutrient advice
* Irrigation control

## Implementation

* Static JSON rules
* Condition-based logic
* No network required
