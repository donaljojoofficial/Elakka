# Frontend Guidelines

## Theme

Dark Forest UI (Idukki jungle aesthetic)

## Colors

* Background: #0d1a0f
* Surface: #132016
* Accent: #4ade80
* Text: #e8f5e9

## UX Principles

* Minimal animations
* High data density
* Fast navigation
* Offline reliability

## Components

* Cards
* Badges
* Bottom navigation
* Modal sheets
